Using Remote Control (Today)

Quick Start

  1. Go to claude.ai/code in your browser (log in with Garen's Anthropic account)
  2. The Mini already has a session running in tmux called rc-taylor
  3. If the session is active, you'll see "Taylor Studio" — click to connect
  4. You're now working on Mini files from your browser. Your project folders are in the sidebar.

If the session needs restarting

Ask Garen to SSH into the Mini and run:

# Attach to existing tmux session $ ssh -t mini "tmux attach -t rc-taylor" # Or start a fresh one $ ssh -t mini "tmux new -s rc-taylor" mini$ cd ~/taylor-studio mini$ claude remote-control --name 'Taylor Studio'

Claude Desktop SSH Setup (Future)

When Taylor has her own Anthropic account

  1. Install Claude Desktop on Taylor's Air
  2. Open Claude Desktop → click the environment dropdown → + Add SSH connection
  3. Fill in:
    Name: Mac Mini
    SSH Host: 100.124.172.108 (Tailscale IP)
    SSH Port: 22
  4. Select project folder: /Users/taylorhopkins/taylor-hopkins-art
  5. Start a session — Claude runs on the Mini with full file access

One-time shell setup needed first

Run on the Mini (as admin) so Taylor's account can find claude:

$ sudo sh -c 'echo "eval \"\$(/opt/homebrew/bin/brew shellenv)\"" >> /Users/taylorhopkins/.zprofile' $ sudo chown taylorhopkins:staff /Users/taylorhopkins/.zprofile
