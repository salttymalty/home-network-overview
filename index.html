How things connect

Everything talks to the Mini. The Mini never sleeps.

NeedHowProtocol
Backup Both Airs back up to Mini automatically Time Machine over SMB
Remote control Screen Share or SSH from either Air vnc://mini.local
Shared files Family folder on Mini, visible to all SMB via /Users/Shared/
Media Mini serves library to Apple TV Home Sharing
Code sync Git push/pull between Air and Mini git over SSH
Heavy compute SSH into Mini, run there ssh garen@mini.local
Print / AirDrop All devices see each other Bonjour / AirDrop

Claude Code across machines

The ecosystem is git-native. Repos sync between machines via GitHub, not file sharing.

Setup sequence

1. Mac Mini (do first)

  1. Clean macOS setup — account garen, personal Apple ID
  2. Create Taylor's standard account
  3. Enable: Screen Sharing, File Sharing, Remote Login (SSH)
  4. Enable: Time Machine sharing (designate a shared volume or folder)
  5. Run bootstrap-mac.sh from USB or clone salttymalty/tools
  6. Complete Google OAuth flows (OEC + personal)
  7. Install Homebrew, ffmpeg, yt-dlp
  8. Install Claude Code
  9. Set Energy Saver: prevent sleep when display is off

2. Garen's Air

  1. Run bootstrap-mac.sh (skip LaunchAgents — server role is Mini only)
  2. Set up SSH key for Mini: ssh-copy-id garen@mini.local
  3. Point Time Machine at Mini's shared volume
  4. Install Claude Code

3. Taylor's Air

  1. Standard macOS setup with Taylor's Apple ID
  2. Point Time Machine at Mini's shared volume
  3. Connect to Mini shared folder in Finder if desired
  4. That's it — her machine, her rules

4. Apple TV

  1. Enable Home Sharing (same Apple ID as Mini, or Family Sharing)
  2. Mini's shared library appears automatically

Design principles

No subscriptions

Everything runs on built-in macOS services. No Dropbox, no NAS OS, no cloud sync fees.

Git is the sync layer

Code moves between machines via GitHub. File sharing is for family stuff, not repos.

Mini never sleeps

It's a server. Backups, cron jobs, and encoding happen whether anyone's at the keyboard or not.
